SDN Campus Project
Description

Projet SDN Campus utilisant POX, IA et automatisation pour superviser et gérer intelligemment un réseau campus.
Permet d’automatiser la configuration des flux, détecter les anomalies et optimiser les performances du réseau.

Objectifs pédagogiques

Centraliser la topologie et la configuration réseau dans Git

Automatiser la gestion des flux SDN via Python et POX

Surveiller le réseau en temps réel avec les scripts de monitoring

Détecter automatiquement les anomalies réseau grâce à un agent IA

Modifier dynamiquement les flux et appliquer des correctifs

Architecture et outils

Contrôleur SDN : POX

Agent IA collaboratif : Python (agent/ai_agent.py)

Automatisation des flux : automation/flow_manager.py

Monitoring : monitoring/monitor.py et monitor_real.py

Topologie réseau : topology.py

Environnement Python : requirements.txtRésultats attendus

Flux réseau gérés et optimisés automatiquement

Détection et traitement des anomalies par l’agent IA

Supervision en temps réel du réseau

Automatisation de certaines tâches SDN via Python
